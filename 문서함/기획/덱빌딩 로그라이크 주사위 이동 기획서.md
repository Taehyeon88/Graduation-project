## 기획 의도

본 프로젝트는 ‘슬레이 더 스파이어’의 카드 선택(What)이라는 검증된 재미에, 주사위와 3D 그리드를 활용한 ‘제약적 이동(Where)’을 결합하여 독창적인 전투 흐름을 만들어내는 것을 목표로 합니다.

플레이어는 매 턴 주사위가 결정해 준 ‘이동 가능 횟수’라는 변수 안에서 최적의 경로를 설계해야 합니다. 특히, 이동하지 않고 제자리에 머무르면 다음 턴의 기동력이 감소하는 독특한 ‘정체 패널티’시스템을 도입하여, 플레이어가 끊임없이 움직이며 전장을 누비도록 유도했습니다.

이를 통해 플레이어는 랜덤하게 배치된 장애물과 자원(코스트) 아이템, 그리고 몬스터의 위협 사이에서 매 순간 생존, 그리고 파밍을 위한 최선의 수를 고민하는 역동적인 플레이를 경험하게 될 것입니다.

## 재미 요소

1. 주사위가 만드는 통제 불가능한 변수  
   * **운과 대처 능력** : 이동력이 고정되어 있지 않고 매 턴 주사위에 의해 결정됩니다. 낮은 숫자가 나왔을 때는 최소한의 움직임으로 위기를 넘기고, 높은 숫자가 나왔을 때는 과감하게 적의 뒤를 잡거나 아이템을 획득하는 등 상황에 따른 유연한 판단이 요구됩니다.  
   * **8방향의 자유와 제약** : 캐릭터를 둘러싼 8방향으로 자유롭게 이동할 수 있지만, 장애물이나 몬스터가 길을 막는 변수가 존재하여 퍼즐 풀듯이 이동 경로를 짜는 재미를 줍니다.

2. 움직여야 사는 리스트 관리 시스템  
   * **적극적 이동 유도** : 이번 턴에 이동 횟수를 남기면 다음 턴의 이동력이 깎이는 리스트 시스템을 통해 플레이어가 안전한 곳에만 머무르는 ‘존버(수동적 플레이)’를 방지합니다.  
   * **전술적 딜레마** : “당장 필요 없어도 억지로 움직여서 다음 턴을 대비할 것인가?”아니면 “다음 턴의 패널티를 감수하고 현재 위치를 사수할 것인가?”를 끊임없이 고민하게 만듭니다.

3. 전장을 활용한 자원 확보  
   * **위험 감수와 보상** : 맵 곳곳에 배치된 랜덤 타일에는 카드 사용 자원(코스트)을 \+1 해주는 아이템이 존재합니다.  
   * **동선 전략** : 플레이어는 단순히 적을 피하는 것을 넘어, 공격을 당할 위험을 무릅쓰고 아이템을 먹으러 갈 것인지 결정해야 합니다. 이는 이동 시스템을 단순히 회피 수단이 아닌, ‘성장과 자원 확보’의 수단으로 확장시킵니다.

4. 카드와 공간의 입체적 연계  
   * **What & Where** : ‘어떤 카드를 쓸까’의 고민에 앞서 ‘주사위 눈만큼 어디로 갈 수 있는가’를 먼저 계산 해야합니다. 카드의 사거리를 고려해 적절한 타일로 이동하고, 장애물을 이용해 적의 공격을 무효화 하는 등 3D 공간을 활용한 전략의 깊이를 제공합니다.

## 게임 조작법

* 마우스 좌클릭 : 선택, 클릭  
* 마우스 우클릭 : 카드 선택 취소  
* 1\~0 : 카드 선택  
* E : 턴 종료  
* A : 캐릭터 이동 모드 진입/해제  
* S : 버린 카드 더미  
* D : 뽑을 카드 더미  
* ESC : 퍼즈 메뉴

## 클리어/패배 조건

* 목표 : 플레이어는 준비된 총 3개의 막(Act)을 모두 생존하여 돌패햐애 합니다.  
* 게임 종료 시점 : 제3막(Act 3)의 최종 보스를 처치하는 순간 게임이 종료되며, 플레이어의 승리로 판정이 됩니다.  
* 패배 조건 : 스테이지 진행 도중 몬스터의 공격이나 함정 등에 의해 플레이어 캐릭터의 체력(HP)이 0이 될 경우, 즉시 패배 처리되며 제1막의 초기 상태로 돌아갑니다.

## 주요 오브젝트

1. 플레이어 캐릭터 (Player Character)  
   * 정의 : 유저가 직접 조작하는 아바타이자, 덱(Deck)의 주체입니다.  
   * 특징:  
     1. 상태 값 : 체력(HP), 방어도(Shield)를  보유합니다.  
     2. 이중 자원 시스템 : 카드 사용을 위한 \[코스트(Mana)\]와 이동을 위한 \[이동력(SPD)\] 두 가지 자원을 보유합니다.  
     3. 자유 행동 : “이동 후 공격”혹은 “공격 후 이동”처럼 순서가 고정되어 있지 않습니다. \[이동 1칸 \-\> 공격 카드 사용 \-\> 다시 이동 2칸\]과 같이 보유한 자원 안에서 이동과 행동을 자유롭게 번갈아 수행할 수 있습니다.  
     4. 디자인 : 로우 폴리 스타일의 캐릭터.  
2. 몬스터 캐릭터  
   * 정의 : 플레이어의 진행을 방해하고 공격하는 적대적 개체입니다.  
   * 특징:  
     1. 행동 패턴 : 공격,방어,버프/디버프 등 매턴 수행할 행동이 머리 위에 아이콘으로 미리 표시됩니다.  
     2. 공격 범위 : 공격할 타일(grid)을 붉은 색으로 미리 시각화하여 플레이어에게 회피의 기회를 제공합니다.  
     3. 충돌 : 타일 위에 존재하며, 플레이어가 이동하려는 경로를 물리적으로 차단하는 ‘장애물’의 역할도 겸합니다.  
3. 카드  
   * 정의 : 플레이어가 수행하는 주요 행동(공격, 방어, 스킬)의 최소 단위입니다.  
   * 구성:  
     1. 코스트 : 사용 시 소모되는 자원량  
     2. 효과 : 데미지, 방어도 획득, 특수 효과, 타일 설치 등  
     3. 사거리 : 3D 그리드 상에서 스킬이 닿는 유효 범위가 설정되어 있습니다.  
   * 역할 : 덱 빌딩의 핵심 요소로, 플레이어는 한정된 코스트 내에서 카드를 조합하여 최적의 효율을 내야합니다.  
4. 주사위  
   * 정의 : 플레이어의 이동력(SPD)을 결정하는 핵심 운(RNG) 요소입니다.  
   * 매커니즘:  
     1. 자원 공급 : 턴 시작 시 주사위를 굴려 나온 숫자만큼 플레이어에게 \[이동력(SPD)\]을 부여합니다.
