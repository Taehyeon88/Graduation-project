## 1\. 버프 \- 이로운 효과

### \[집중 N\]

* **분류** : 기간제 효과  
* **효과 설명** : N턴 동안 공격력이 25% 증가합니다.  
* **상세 기획 및 연산 규칙**:  
  * 해당 버프는 횟수가 아닌 턴 수로 진행되며, 공격 횟수에는 제한이 존재하지 않는다.  
  * 턴 종료 시 감소 핵심 로직을 따른다.  
  * 직접 피격을 주는 객체에만 적용되며, 영역/틱 대미지/고정 대미지에는 영향을 줄 수 없다.  
  * 공격력 증가는 \[공격자 계수\]에 포함되어 계산된다.  
    * ‘대미지 연산 시스템’ 문서 ‘3.3. 공격자 계수’ 참조  
    * [대미지 연산 시스템](https://docs.google.com/document/d/1Zi_XVD9c8OBOyNFwyo_VRaFTP4d1t2kTjq2t21_uPmQ/edit?usp=sharing)

### \[방어도\]

(※ N 수치 없음)

* **분류** : 조건제 효과  
* **효과 설명** : 임시 추가 체력을 얻어 피해를 우선 차감한다.  
* **상세 기획 및 연산 규칙**:  
  * 턴 종료 시 감소 핵심 로직을 따른다.  
  * 턴 수(N)를 가지지 않으며, 기본적으로 다음 턴 시작 시 0으로 초기화된다.  
    * \[틱 피해\] 계열 보다 먼저 진행 된다.  
  * 단, 유물이나 특정 카드 효과(전용/능력)에 따라 초기화가 진행되지 않거나 특정 수치가 보존될 수 있다.  
  * 방어도가 보존 될 경우 (유물/특정 카드 효과 적용 시)  
    * 중독/출혈 등의 틱 피해 발생 시 : 방어도를 뺀 최대 체력의 n%를 계산한 후, 도출된 피해량을 방어도에서 우선 차감하고 남은 피해를 본체에 적용된다.

### \[차징 N\]

* **분류** : 조건제 효과  
* **효과 설명** : N턴 내에 플레이어의 턴 종료 시 \[인접 1칸\] 내에 몬스터가 존재하고 방어도가 1 이상이면, 내 \[인접 1칸\]의 모든 적을 1칸 밀쳐낸다.  
* **상세 기획 및 연산 규칙**:  
  * 턴 종료 시 감소 핵심 로직을 따른다.  
  * 해당 조건은 몬스터의 턴 종료와 플레이어의 턴 시작 사이에 계산을 하며, 플레이어가 방어도가 1 이상 남을 시 발동한다.  
  * 해당 조건 달성되면 객체 기준 인접 1칸 내의 모든 적을 뒤로 1칸 밀어내며, 넉백으로 인한 충돌 발생 시 충돌 피해를 입힌다.  
  * 주변에 적이 없어 효과가 발동하지 않은 경우에는 N 수치만 1 감소한다.  
  * \[조건제 절대 규칙 적용\] : 넉백 효과가 발동했다면, 남은 N 수치와 상관없이 즉시 해당 상태효과는 제거된다.

## 2\. 디버프 \- 해로운 효과 / 상태 이상

### \[혼란 N\]

* **분류** : 기간제 효과  
* **효과 설명** : N턴 동안 공격력이 50% 감소한다.  
* **상세 기획 및 연산 규칙**:  
  * 해당 디버프는 횟수가 아닌 턴 수로 진행되며, 공격 횟수에는 제한이 존재하지 않는다.  
  * 턴 종료 시 감소 핵심 로직을 따른다.  
  * 직접 피격을 주는 개체에만 적용되며, 영역/틱 대미지/고정 대미지에는 영향을 주지 않는다.  
  * 공격력 감소는 \[공격자 계수\]에 포함되어 계산된다.  
    * ‘대미지 연산 시스템’ 문서 ‘3.3. 공격자 계수’ 참조  
    * [대미지 연산 시스템](https://docs.google.com/document/d/1Zi_XVD9c8OBOyNFwyo_VRaFTP4d1t2kTjq2t21_uPmQ/edit?usp=sharing)

### \[표적 N\]

* **분류** : 기간제 효과  
* **효과 설명** : N턴 동안 받는 피해가 50% 증가한다.  
* **상세 기획 및 연산 규칙**:  
  * 턴 종료 시 감소 핵심 로직을 따른다.  
  * 횟수가 아닌 턴 수로 진행되며, 피격 횟수에는 제한이 존재하지 않는다.  
  * 직접 피격을 받는 효과이므로 일반 피격과 영역 피해에 모두 적용된다. 단, 틱 데미지와 고정 데미지에는 영향을 주지 않는다.  
  * 받는 피해 증가는 \[방어자 계수\]에 포함되어 계산된다.  
    * ‘대미지 연산 시스템’ 문서 ‘3.4. 방어자 계수’ 참조  
    * [대미지 연산 시스템](https://docs.google.com/document/d/1Zi_XVD9c8OBOyNFwyo_VRaFTP4d1t2kTjq2t21_uPmQ/edit?usp=sharing)

### \[부상 N\]

* **분류** : 기간제 효과  
* **효과 설명** : N턴 동안 턴 시작 시, 손패 카드 중 랜덤 1장의 코스트가 증가한다.  
* **상세 기획 및 연산 규칙**:  
  * 턴 종료 시 감소 핵심 로직을 따른다.  
  * 발동 시점 : 턴 시작 단계에서 기본 드로우로 손태에 카드가 들어온 직후 적용된다.  
    * 카드 효과로 인해 전투 중 덱에서 추가로 뽑는 카드에는 적용되지 않는다.  
  * 모두 동일한 확률로 무작위 1장이 지정된다.  
  * UI/UX: 증가된 코스트의 카드는 코스트 표기 부분에 배경으로 붉은색으로 적용하여 직관적으로 보여주어야 한다.  
  * 증가된 코스트는 카드를 사용하거나, 턴이 종료되면 원래의 코스트로 돌아간다.

### \[출혈 N\]

* **분류** : 기간제 효과  
* **발동 시점** : 턴 시작 시  
* **효과 설명** : N턴 동안 턴 시작 시 최대 체력의 3% 피해를 입는다.  
* **상세 기획 및 연산 규칙**:  
  * 턴 종료 시 감소 핵심 로직을 따른다.  
  * \[틱 피해\]로 지칭하며, 턴 시작 시 방어도가 0으로 초기화된 직후에 발동한다.  
    * 만약 방어도가 유물/효과 등으로 보존되어 남아있다면, 산출된 피해량을 방어도에서 우선 차감한 뒤 남은 피해를 본체에 가한다.  
  * 피해량 기준은 대상 객체의 ‘현재 체력’이 아닌 ‘최대 체력’이다.  
  * 상세 대미지 계산식은 ‘대미지 연산 시스템’ 문서 ‘4.1. 턴 시작 상태 이상 피해’ 참조

## 3\. 군중 제어기 (CC/제어 효과)

### \[고립 N\]

* **분류** : 기간제 효과  
* **효과 설명** : N턴 동안 자발적 이동 및 이동 관련 카드의 사용이 불가하다.  
* **상세 기획 및 연산 규칙**:  
  * 턴 종료 시 감소 핵심 로직을 따른다.  
  * UI/UX: 해당 지속 중에는 손패의 이동 관련 카드를 회색으로 어둡게(비활성화) 처리하여 사용 불가 상태임을 직관적으로 표기한다.  
  * 자발적인 이동만 봉인될 뿐, 적으로부터의 밀쳐짐(넉백)이나 끌어당겨짐 등 외부 객체에 의한 강제 이동에는 정상적으로 영향을 받는다.

### \[행동 불가 N\]

* **분류** : 기간제 효과 (턴 스킵)  
* **효과 설명** : N턴 동안 이동 및 모든 스킬/카드 사용이 불가하다.  
* **상세 기획 및 연산 규칙**:  
  * 턴 종료 시 감소 핵심 로직을 따른다.  
  * 발동 조건 : 턴 시작 시 가장 먼저 확인되며, 존재할 경우 즉시 턴을 스킵한다.  
    * 플레이어 : 턴 스킵 시 손패에 새로운 카드가 들어오지 않고 \[틱 피해\], \[방어도 초기화\]는 진행 후 바로 몬스터 턴으로 넘어간다.

### \[집중 N\]

* **분류** : 기간제 효과  
* **발동 조건** : 넉백으로 인한 충돌 발생시  
* **효과 설명** : N턴 내에 밀쳐져서 충돌할 경우, 즉시 \[행동 불가\] 상태로 변환된다.  
* **상세 기획 및 연산 규칙**:  
  * 턴 종료 시 감소 핵심 로직을 따른다.  
  * \[조건제 절대 규칙 적용\] : N턴이 끝나기 전에 충돌 조건을 달성하면, 남은 N 수치와 상관없이 즉시 해당 \[압사 대기\] 효과가 제거되고, 기획된 \[행동 불가\] 효과로 상태가 넘어간다.  
  * 넉백으로 인한 충돌 상황 외에는 발동하지 않는다.  
  * 조건을 달성하지 못한 채 턴 종료 감소 로직을 타서 N 수치가 모두 소모(0)되면 효과는 단순히 제거된다.
