## **1\. 개요**

스킬 로직 및 타겟팅/예외 처리 구현을 위한 상세 명세서입니다.

### 참고 상태 효과:

* \[집중 N\] : N턴 동안 공격력 25% 증가  
* \[혼란 N\] : N턴 동안 공격력 50% 감소  
* \[고립 N\] : N턴 동안 이동 불가, 이동 관련된 카드 사용 불가  
* \[표적 N\] : N턴 동안 받는 피해 50% 증가  
* \[방어도\] : 임시 추가 체력 (피해 우선 차감)  
* \[출혈 N\] : N턴 동안 턴 시작 시 최대 체력의 3% 피해  
* \[중독 N\] : N턴 동안 턴 시작 시 최대 체력의 3% 피해  
* \[악화\] : 대상에게 부여된 \[중독\] 등의 상태 이상 효과 증폭 및 연계  
* \[부상 N\] : N턴 동안 손패 카드 중 랜덤으로 1장의 소모 마나(코스트) 1증가  
* \[압사 대기 N\] : N턴 동안 밀쳐져서 충돌 시 \[행동 불가 N\] 부여  
* \[행동 불가 N\] : N턴 동안 이동 불가, 몬스터의 공격 및 스킬 행동 불가  
* \[차징\] : 다음 턴 시작 시 방어도가 1 이상이면 내 \[인접 1칸\]의 모든 적 1칸 밀치기

2. 타겟을 지정하여 공격할 때, 지정된 타겟의 바로 뒤 타일이 벽인지 검사하여 0 또는 1 코스트를 차감합니다.

* 수치 데이터:

  * 조건부 딜링기 : 적을 벽으로 몰아넣은 상태에서 콤보 후속타로 마나 소모 없이 딜을 누적시키는 핵심 서브 공격기입니다.  
  * 이중 코스트 체크 : 플레이어 주변에 뒤가 벽인 적(A)과 뒤가 뚫려있는 적(B)이 둘 다 존재할 경우 UI상으로는 1 코스트를 표시하지만 플레이어가 적 A를 타겟팅할 경우 코스트는 0으로 변경되어있어야하고 사용 할 시 정상적으로 차감  
  * 플레이어의 이동, 적의 이동, 장애물 생성/파괴 등 전장 상황이 바뀔 때마다 손패에 있는 이 카드의 조건 달성 여부를 검사하여 코스트 표기를 실시간으로 업데이트해야 합니다.  
  * 0코스트로 표기된 상태에서 \[부상\] 디버프에 의해 코스트가 1 증가할 경우 해당 코스트는 1로 표기  
    * 기존 : 0코스트  
    * 부상 효과 : 0코스트 \+ 1(부상)코스트 \= 1코스트

   * 타일이 비어있을 경우 : 플레이어가 1칸 전진합니다.  
   * 적 몬스터가 있을 경우 (제설차 & 박치기) : 대상 적의 바로 뒤 타일(밀려날 곳) 상태를 확인합니다.  
     * 뒤가 뚫려 있는 경우 (제설차) : 대상을 1칸 밀쳐내고 4피해를 줍니다. 그리고 플레이어가 1칸 전진합니다.  
     * 뒤가 막혀 있는 경우 (박치기) : 대상은 밀려나지 않고 제자리에서 \[벽쿵 충돌 이벤트 \+ 불도저 피해\]를 한 번에 받습니다. 이때 플레이어는 전진하지 않고 제자리에 머물며, 스킬이 취소되지 않고 곱바로 2번째 판정으로 넘어갑니다.  
3. \[2번째 판정\]  
   * 1전째 판정이 끝난 현재 위치에서 동일한 방향으로 한 번 더 앞선 판정 로직을 반복합니다.  
   * 뒤가 막힌 적에게 박치기를 했을 경우 : 플레이어와 적 모두 제자리이므로, 또다시 \[벽쿵 충돌 이벤트 \+ 불도저 피해\]가 연달아 터지게 됩니다. (제설차처럼 밀면서 전진했을 경우엔 밀려난 적을 따라가며 2번째 넉백 및 피해를 줍니다.)  
* 수치 데이터:  
  * 피해량 : 4+4  
  * 사거리 : 2  
  * 공격 범위(타일 수) : 2  
  * 본인에게 효과 부여 : X  
  * 적에게 효과 부여 : X  
* 예외 처리 & 특이사항:  
  * \[장판 상호작용\]  
    * **몬스터** : 제설차처럼 밀쳐져서 도착한 타일(혹은 거쳐가는 타일)에 ‘적에게 해로운 장판’이 깔려있을 경우, 몬스터는 해당 **장판의 진입 피해 및 상태이상(디버프)을 즉시 적용받음**.  
      * 해당 효과에 의해 데미지가 달라질 수 있음  
    * **플레이어** : 적을 밀어낸 후(또는 빈타일로) 플레이어가 전진하여 도착한 타일이 장판일 경우, 플레이어에게 면역이 없다면 **플레이어 역시 장판의 진입 피해 및 디버프를 동일하게 받음**. (밀고 나가다가 자해할 수 있는 하이리스크 이동기)  
      * 해당 효과에 의해 데미지가 달라질 수 있음  
  * \[고립(이동불가), 행동 불가 특수 판정\]  
    * **몬스터** : 불도저에 읜해 밀려난 몬스터가 \[고립\]을 부여하는 함정/장판(예: 거미줄 함정)에 닿을 경우, 즉시 \[고립\] 상태이상 표식이 생기지만 불도저의 강력한 강제 이동(넉백) 물리력이 우선 판정되어 **몬스터는 이동 불가를 무시하고 계속 밀려남**.  
    * **플레이어** : 반면, 플레이어가 1번째 이동 판정을 통해 ‘플레이어에게 \[고립\]을 부여하는 장판’에 진입하게 된다면, 플레이어는 즉시 \[고립\] 상태에 걸림과 동시에 **불도저 스킬의 2번째 이동 및 타격 판정이 그 즉시 완전히 취소됩니다.** (발이 묶여 더 이상 돌진 불가능)  
    * 만약 1번째 넉백 데미지(혹은 장판 데미지)로 인해 **적이 사망 했다면**, 바로 이어지는 2번째 판정에서 ‘적 몬스터가 있을 경우’의 로직이 아닌 타일이 비어있을 경우의 로직을 타서 플레이어만 1칸 빈 공간으로 전진(추가 데미지 없음)하게 됩니다.  
    * \[**핵심 시너지 \- 2연속 벽쿵**\] : 이 카드의 진가는 적을 벽에 몰아넣고 썼을 때 발휘됩니다. 대상이 뒤로 밀리지 못하므로 2코스트로 ‘**벽쿵 2번, 8피해(4+4)**’라는 어마어마한 연속 누킹이 가능해집니다.
