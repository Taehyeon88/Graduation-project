# 스킬

ID: PN-1
목록: 카드
버전: 0.0.1

## 스킬 정의

정의 : 플레이어가 공격/유틸리티를 소유하고 있는 AP(action point)를 소모해서 사용

스킬 타입:

공격:

일반 : 일반적인 공격

- 플레이어는 기본적인 공격을 하는 순수한 타입
- 인접 단일, 횡베기, 전방위, 관통, 전후방 등 여러 공격 타입으로 존재

넉백 : 적을 밀어냄

- 공격과 넉백이 결합된 상태.
- 일반 상태의 넉백 효과가 추가된 느낌추가 피해 발생
- 넉백으로 인한 충돌이 발생시

신성(아이덴티티) : 적에게 표식 부여

- 공격과 신성이 결합된 상태.
- 일반 상태에 신성 효과가 추가된 느낌
- 피격을 통해 해당 효과를 부여함.
- 신성 종류:
    - 폭파의 신성 : 신성이 부여된 적을 신성효과가 붙지 않는 스킬로 피격 시 터지며 인접 1칸 내 적에게 피해를 줌
    - 치유의 신성 : 몬스터를 처치 시 해당 몬스터에게 부여된 신성 만큼 회복

유틸리티:

이동 : 캐릭터 이동

- 캐릭터의 이동을 담당
- 기본 이동과는 다르게 많이 이동하거나 특정 효과를 가지고 있음
- 피해를 주는 효과가 결합된 것도 존재

당기기 : 적을 당김

- 몬스터를 플레이어 앞으로 당기는 효과

위치교환 : 적과 위치를 교환

- 인접 1칸 내 존재하는 적과의 위치를 교환할 수 있음
- 중반에 등장하는 카드
- 상점이나 이벤트에서 획득 가능

방어도 : 플레이어 캐릭터에 임시 체력 부여

- 캐릭터에게 임시 체력을 부여
- 신성이 표식된 적 수량에 따라 획득하는 방어도 수치가 많아질 수 있음

## 효과 부여

정의 : 공격 스킬에 일반적인 효과를 부여하여 부가 효과를 보는 형태

## 스킬 AP 자원 밸런싱 설계

- 0 AP (프리 행동) : 단순 1칸 이동(기본 이동과 다름), 쿨타임이 존재하는 스킬
- 1 AP (기본 행동) : 약한 공격, 짧은 대쉬, 방어력 획득. 한 턴에 2번 사용하여 템포를 유지할 수 있음
- 2AP (전술 행동) : 넉백, 상태 이상 부여, 중간 범위 공격. 들어온 AP를 모두 소진하게 됨
- 3~5 AP (필살기/궁극기) : 십자형, X자형 등 넓은 범위를 타격하거나 치명적인 대미지를 줌. 이 스킬을 쓰려면 이던 턴에 스킬 사용을 아끼고 AP를 모으는 리스크를 감소해야 함.
