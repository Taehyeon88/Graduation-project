# 공격

#### 방패 밀치기

- 효과 : 인접한 1칸의 적에게 하트 ****피해를 주고, 해당 적을 내 공격 방향으로 1칸 밀쳐냅니다. 밀려난 자리에 벽이나 다른 적이 있다면 추가로 하트 ****피해를 줍니다.
- 범위 : 인접1/단일

#### 휩쓸기

- 효과 : 플레이어를 중심으로 한 주변 8방향의 모든 적에게 하트 ****피해를 줍니다.
- 범위 : 인접1/전방위

#### 고립 타격

- 효과 : 인접한 적 1명에게 공격합니다. 만약 타겟 인접1칸 이내에 다른 몬스터가 한 마리도 없다면, 피해 **2**를 줍니다.
- 범위 : 인접1/단일

#### 갈고리 사슬

- 효과 : 상하좌우 직선상으로 최대 2칸 거리에 있는 적에게 피해 1을 주고, 내 바로 앞 빈칸으로 강제로 끌고 옵니다.
- 범위 : 인접2/직선

#### 시한 타격

- 효과 : 비어있는 타일 1곳을 지정해 공격 예고 마커를 찍습니다. 다음 턴이 시작될 때 해당 타일과 그 십자 인접 타일에 있는 모든 개체에게 피해 2를 줍니다.
- 범위 : 인접2/십자

# 방어

#### 지형엄폐

- 효과 : 방어도를 얻습니다. 플레이어와 단일1에 '설치물'이나 '맵의 벽'이 있다면 추가로 방어막 **1**개를 더 얻습니다.
- 대상 : 본인
- 유지 턴 : 1턴

#### 환영 방패

- 효과 : 이번 턴에 날라오는 적의 투사체 공격을 2회 무효화 시킵니다.
- 대상 : 본인
- 유지 턴 : 1턴
- 무효화 대상 : 투사체 공격

#### 빙결의 장갑

- 효과 : 방어도 1을 얻습니다. 이번 턴에 방어도가 유지된 상태로 피격될 시 피격된 대상은 기절을 하게 됩니다.
- 대상 : 본인
- 유지 턴 : 1턴
- 기절 대상 : 피격한 적

#### 혈투의 장막

- 효과 : 체력 1칸을 소모하여 방어도 3을 얻습니다.
- 대상 : 본인
- 유지 턴 : 1턴

#### 임기응변

- 효과 : 카드 1장을 버리고 방어도를 얻습니다.
- 대상 : 본인
- 유지 턴 : 1턴

# 설치

#### 자력 탑

- 효과 : 인접2칸 내 모든 적을 1칸씩 끌어당깁니다.
- 대상 : 몬스터
- 체력 : 2
- 유지 턴 : 2

#### 가시 함정

- 효과 : 가시 함정을 설치합니다. 밟을 시 즉시 이동을 멈추고 피해를 줍니다.
- 대상 : 몬스터
- 체력 : 1
- 유지턴 : 3

#### 연막

- 효과 : 연막을 설치합니다. 연막 안에 있을 시 원거리 타겟팅을 할 수 없습니다.
- 대상 : 플레이어
- 유지턴 : 2

#### 수호의 깃발

- 효과 : 수호의 깃발을 설치합니다. 깃발 인접 1칸 범위로 안전범위가 생성되며, 받는 모든피해가 1로 고정됩니다.
- 대상 : 플레이어
- 유지턴 : 2
- 내구도 : 3회

#### 마력 흡수기

- 효과 : 마력 흡수기를 설치합니다. 흡수기 인접 1칸 범위로 퍼지며, 범위 내에서 적을 처치 시 코스트 1을 획득합니다.
- 대상 : 플레이어
- 유지 턴 : 2
- 내구도 : 3회
