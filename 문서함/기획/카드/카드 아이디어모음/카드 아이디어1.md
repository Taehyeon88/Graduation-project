# 공격 카드

#### 십자 포화

- 효과 : 지정한 적 1명과 그 상화좌우 인접 타일에 있는 모든 적에게 피해를 줍니다.
- 범위 : 인접 1/단일

#### 관통격

- 효과 : 직선상에 있는 적 2명에게 피해를 줍니다. 앞에 적이 있다면, 뒤에 있는 적에게 2 피해를 줍니다.
- 범위 : 인접 2/관통

#### 처형(임시)

- 효과 : 단일 적에게 큰 피해를 줍니다. 이 공격으로 적이 처치된다면, 이번 턴에 사용한 코스트의 절반을 돌려받습니다.
- 범위 : 인접 1/단일

#### 연쇄 번개

- 효과 : 대상에게 피해를 주고, 인접 1타일 이내에 다른 적이 있다면 최대 2명에게 무작위로 튕기며 피해를 줍니다.
- 범위 : 인접 1/단일

#### 약점 공격

- 효과 : 방어도를 무시하고, 피해를 줍니다.
- 범위 : 인접 1/단일

# 방어

#### 굳건한 방진

- 효과 : 방어도를 얻습니다. 얻은 방어도는 다음 턴 시작 시 사라지지 않고 50%가 유지됩니다.
- 대상 : 본인

#### 반격 태세

- 효과 : 방어도를 얻습니다. 방어도를 잃지 않고 다음 턴이 올 경우 코스트 +1 추가 획득합니다.
- 대상 : 본인

#### 충격 흡수

- 효과 : 방어도를 얻습니다. 방어도를 전부 소모할 경우, 다음 턴 첫 카드에 추가 피해 +1이 부여됩니다.
- 대상 : 본인

#### 기동 회피

- 효과 : 이번 턴에 이동을 할 때마다 방어도를 획득합니다.
- 대상 : 본인

#### 가시 갑옷

- 효과 : 방어도를 얻습니다. 이번 턴 동안 방어도가 존재할 때 피격 시 적에게 동일한 피해를 줍니다.
- 대상 : 본인

# 유틸

#### 전술적 재장전

- 효과 : 손패에서 카드를 2장 버리고, 3장을 새로 뽑습니다.
- 대상 : 본인

#### 과부하

- 효과 : 코스트를 2 얻습니다. 단 다음 턴에 획득할 카드 1장 감소합니다.
- 대상 : 본인

#### 시선 교란

- 효과 : 지정한 적 1명의 공격 방향을 반대로 돌립니다.
- 대상 : 적

#### 집중

- 효과 : 이번 턴에 다음에 사용하는 공격 카드의 추가 피해 +1을 부여합니다.
- 대상 : 본인

#### 설치물 폭파

- 효과 : 설치된 설치물 중 1캐를 선택하여 폭파 시켜 피해를 줍니다.
- 대상 : 설치물

# 설치

#### 덫 매설

- 효과 : 빈 타일에 지뢰를 설치합니다. 적이 해당 타일을 밟으면 기절과 피해를 줍니다,
- 설치 범위 : 인접 2/단일
- 대상 : 적

#### 미끼

- 효과 : 미끼를 설치합니다. 공격 대상 우선 순위가 최우선으로 타겟을 삼아 어그로가 끌립니다.
- 설치 범위 : 인접 2/단일
- 어그로 범위 : 인접 2
- 체력 : 1
- 대상 : 적

#### 바리스타

- 효과 : 바리스타 포탑을 설치합니다. 매 턴 종료 시 바라보는 직선 범위로 피해를 줍니다.
- 설치 범위 : 인접 2/단일
- 공격 범위 : 인접 2/관통
- 체력 : 2
- 대상 : 적

#### 바리케이드

- 효과 : 내구도 3칸짜리 장애물을 설치합니다. 적의 이동 경로를 막고, 인접 공격을 막습니다.
- 설치 범위 : 인접 1/단일
- 체력 : 3

#### 충격장

- 효과 : 충격 포탑을 설치합니다. 매 턴 종료 시 인접 1 범위로 피해를 줍니다.
- 설치 범위 : 인접 2/단일
- 공격 범위 : 인접 1/전방위
- 체력 : 2
- 대상 : 적
