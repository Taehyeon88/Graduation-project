# 카드 시스템 및 종류 기획서

## 1\. 개요

게임 내 전투와 전략의 핵심인 카드 시스템의 종류와 메커니즘을 정의한다.  
카드는 크게 공격, 기술, 파워 3가지 유형으로 분류되며, 각 유형은 고유한 사용 방식과 타겟팅 규칙을 가진다.

## 2\. 카드 유형별 상세

### 2.1. 공격 카드

몬스터에게 직접적인 데미지를 입히는 카드군. 공격 범위와 방식에 따라 3가지 하위 타입으로 나뉜다.  
A. 인접 카드 (근접 공격)

* 정의 : 플레이어와 물리적으로 가까운 적을 타격하는 근접 공격.  
* 범위:   
  * 기본:   
    * **플레이어 기준 1타일 \[8방향(상하좌우 \+ 대각선)\]**  
    * 무기를 뻗어 닿는 최대 **2타일**.  
  * 확장 : 원하는 방향에 여러 타일 공격  
  * 돌진 : 공격 전 **이동**이 포함된 공격.  
* 특징:  
  * 가장 기초적인 공격 수단.  
  * \[확장 범위\] 카드를 사용하면 적의 공격 범위 밖에서 안전하게 공격이 가능.  
  * \[돌진\] 카드는 별도의 \[이동 카드\] 소모 없이도 적에게 접근하며 공격 할 수 있음. 덱 압축과 행동력 효율에 좋음.  
  * 타겟을 지정하는 단일 공격 외에도, 주변 1타일(8타일)을 동시에 타격하는 카드를 통해 포위 상황을 타개하거나 모든 적을 공격하는 역할을 함.  
  * 근접 공격의 특성상 적을 밀쳐내어 벽에 부딪히게 하거나, 함정(장판) 타일로 밀어 넣어 전략적으로 구성 가능.  
* 사용 조건 : 사거리(1칸) 내에 적이 있어야 유효 타겟팅 가능.

B. 투사 카드 (원거리 공격)

* 정의 : 멀리 떨어진 적을 저격하거나 관통하는 원거리 공격.  
* 범위 : 직선상 타일 혹은 특정 사거리(2칸)의 타일.  
* 특징:  
  * 약화 특화 : 적의 발을 묶거나(속박), 방어 태세를 무너뜨리는(방어력 감소) 효과를 부여함.  
  * 장애물 판정:  
    * 단일 타격 : 벽을 무시하고 벽 뒤의 적도 공격 가능.  
    * 관통 : 직선 방향 2칸을 타격하나, 경로상에 벽이 있으면 막힘.

C. 영역 카드 (장판)

* 정의 : 타일에 장판을 설치하여 지속적인 영향을 주는 카드.  
* 대상 : 몬스터가 있는 타일 또는 빈 타일 모두 사용 가능.  
* 메커니즘 (양날의 검):  
  * 즉시 발동 : 설치 순간 해당 위치에 1차 효과 발동.  
  * 지속 효과 : 턴 시작 시 해당 타일 위의 대상에게 효과 부여.  
  * 피아식별 없음 : 대부분의 공격력 장판은 플레이어가 밟아도 해로운 효과(공격력 감소, 데미지 등)을 입음. (전술적 위치 선정 중요)  
* 시각 효과 : 타일 시각 효과 기획서의 위험 지역 규칙(점선 붉은 테두리)을 따름.

### 2.2. 기술 카드

전투를 보조하거나 전황을 유리하게 만드는 카드군. 버프, 디버프, 이동 카드로 나뉜다.  
A. 버프 카드 (이로운 효과)

* 정의 : 플레이어 자신을 강화하는 카드.  
* 사용 방식:  
  * 타겟팅 없음 : 카드를 화면 상단(필드)으로 드래그 앤 드롭 하면 즉시 발동.  
* 주요 효과:  
  * 방어도 : 플레이어에게 방어도(추가 체력)을 부여.  
  * 화력 : 공격력을 증가.  
  * 피해 감소 : 받는 피해 감소.  
* 지속 시간 규칙 (스택 \= 턴):  
  * 효과 옆에 붙은 숫자는 **유지되는 턴 수**를 의미한다.  
    * 예 : 화력 2 \= 공격력 증가 효과가 2턴 동안 유지됨.  
  * 차감 : 플레이어 턴 시작 시 숫자가 1씩 감소하며, 0이 되면 효과 소멸.

B. 디버프 카드 (해로운 효과)

* 정의 : 적을 약화하거나 행동을 제약하는 카드.  
* 사용 방식:  
  * 타겟팅 필수 : 반드시 공격 가능 범위(사거리) 내에 몬스터가 존재해야만 사용 가능.  
  * 빈 땅에 사용 시도 시 사용 취소 됨.  
* 주요 효과 :   
  * 속박 : 해당 캐릭터가 이동 불가 상태가 됨.  
  * 노출 : 받는 피해 증가.  
* 지속 시간 규칙 (스택 \= 턴):  
  * 효과 옆에 붙은 숫자는 **유지되는 턴 수**를 의미한다.  
    * 예 : 속박 2 \= 이동 불가가 2턴 동안 유지됨.  
  * 차감 : 플레이어 턴 종료 시 숫자가 1씩 감소하며, 0이 되면 효과 소멸.

C. 이동 카드 (이로운 효과)

* 정의 : \[필수 이동(1칸)\]을 마친 후, 추가적으로 위치를 변경하는 카드.  
* 사용 방식:  
1. 카드 선택 시 이동 가능한 범위 (파랑 원) 표시.  
2. 원하는 타일을 클릭하여 이동.  
* 특징:  
  * 카드마다 이동 패턴이 다름.  
    * 예 : 직선 2칸 이동, 대각선 이동.  
  * 이동 시스템 기획서의 3단계: 카드 이동 규칙을 따름.  
    * [이동 및 주사위 시스템](https://docs.google.com/document/d/1JSvcy2z_o25pktUwD_owAITieYNSg34Fme4SO0fWvfg/edit?usp=sharing) 참조.

### 2.3. 파워 카드 (패시브)

* 정의 : 전투의 흐름을 바꾸는 **지속형 패시브**를 부여하는 카드.  
* 메커니즘:  
  * 전투 지속 : 사용 시 해당 전투가 끝날 때까지 효과가 영구 유지됨.  
  * 소멸 : 사용 후 덱에서 제거되어, 해당 전투 동안 다시 뽑을 수 없음  
    * 버린 카드 더미로 가지 않음  
  * 해제 : 전투 종료(승리/도주/패배) 시 플레이어에게 걸린 패시브 효과는 자동으로 사라짐.  
* 사용 방식 : 버프 카드와 동일하게 타겟팅 없이 사용
