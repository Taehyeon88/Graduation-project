# 대미지 연산 시스템

---

최신 업데이트 : 2026.03.30  
버전 : 0.0.1  
---

## 1\. 개요

게임 내 발생하는 모든 대미지의 산출 및 적용 방식을 정의합니다. 버프, 디버프, 카드 효과 등 다양한 변수가 충돌 없이 적용되도록 연산의 우선순위와 소수점 처리 방식을 엄격하게 통제합니다.

## 2\. 대미지 타입 분류

시스템 내의 피해는 다음 3가지 타입으로 분류되며, 각각 별도의 연산 파이프라인을 가집니다.

1. 스킬 피해 : 카드를 사용하여 직접 가하는 피해 (증감 계수 적용 O)  
2. 고정/충돌 피해 : 넉백, 충돌, 특정 버프에 의해 발생하는 고정 피해 (증감 계수 적용 X)  
3. 상태이상 피해 : \[출혈\], \[중독\] 등에 의해 턴 시작 시 발생하는 최대 체력 비례 피해 (전용 증감 계수 적용)

## 3\. 핵심 산출 공식 (카드 피해)

카드 사용 시 발생하는 스킬 피해는 아래의 단일 공식을 표준으로 사용합니다.

공식 : **최종 피해량 \= Ceil(Max(0, 기본 피해량 \+ 고정 수치 증감) x 공격자 계수 ) x 방어자계수**

(※ Ceil() : 괄호 안의 최종 결과값에서 발생하는 소수점을 무조건 올림 처리하여 정수로 반환)  
(※ Max(0, A) : 괄호 안 값 중 더 큰 값을 선택(반환)처리하여 최소 0으로 처리)

### 3.1. 기본 피해량

* 고정형 : 카드에 명시된 고정 수치  
* 동적형 : 특정 스탯에 비례하는 수치. 연산 중 소수점 발생 시 올림(Ceil) 처리  
  * 예시: \[육중한 압박\] \= Ceil(현재 방어도 \* 1.5)  
* 조건부 추가 피해 합산 : 특정 조건을 만족하여 발생하는 추가 피해는 기본 피해량에 합산되어 한 번의 타격으로 계산됩니다.  
  * 예시 : 기본 피해 4, 조건 달성 시 추가 피해 8인 카드에서 조건 달성 \-\> 기본 피헤량을 12(4+8)로 취급하여 단일 연산

### 3.2. 고정 수치 증감

비율(%) 기반의 계수가 곱해지기 전에 기본 피해량에 직접 더해지거나 뼤지는 고정 수치입니다. (기본값 0\)

* 적용 대상:  
  * 증가(+) : 장비/유물에 의한 기본 공격력 \+N 증가, 특정 능력 카드 사용으로 인한 스킬 피해량 \+N 증가 등  
  * 감소(-) : 특정 적의 방어 기믹에 의한 받는 피해량 \-N 감소, 디버프에 의한 주는 피해량 \-N 감소 등  
* 목적 : 계수 연산 전에 적용되는 \[집중\], \[혼란\] 등의 퍼센트 버프/디버프 효율에 직접적인 영향을 줍니다.  
* 예외 처리 (0 미만 방지) : 기본 피해량에서 고정 수치를 뺀 결과값이 0 미만(음수)이 될 경우, 무조건 0으로 보정한 뒤 다음 퍼센트 계수 연산으로 넘어가야 합니다.

### 3.3. 공격자 계수

공격자가 가진 딜링 관련 버프/디버프의 비율 총합입니다. (기본값 1.0)

* 연산 방식 : 합연산  
* 수식 : Max(0, 1.0 \+ 공격자 버프 총합 \- 공격자 디버프 총합)  
* 해당 요소:  
  * \[집중\] : \+0.25 (25 증가)  
  * \[혼란\] : \-0.50 (50% 감소)  
  * \[견제 사격 : 다음인접카드 사용시\] : \+0.50 (50% 증가)

### 3.4. 방어자 계수

방어자가 가진 피격 관련 버프/디버프으 비율 총합입니다. (기본값 1.0)

* 연산 방식 : 합연산  
* 수식 : Max(0, 1.0 \+ 방어자 디버프 총합 \- 방어자 버프 총합)  
* 해당 요소:  
  * \[표적\] : \+0.50(50% 증가)

## 4\. 특수 피해 연산 산출 공식

일반적인 공격/방어 계수를 타지 않는 예외적인 대미지 산출식입니다.

### 4.1. 턴 시작 상태이상 피해

* 대상 상태이상 : \[출혈\], \[중독\]  
* 공식 : 틱 피해량 \= Ceil(타겟 최대 체력 \* 0.03 \* 특수 증폭 계수)  
* 특수 증폭 계수:  
  * 기본 값 : 1.0  
  * 타겟에게 \[악화\] 디버프 존재 시 2.0 (2배 증폭)

### 4.2. 고정 및 충돌 피해

* 공식 : 최종 피해량 \= 명시된 고정 수치  
* 적용 대상:  
  * 넉백 충돌(벽쿵) 피해  
  * \[공성 전차\] 버프에 의한 충돌 시 추가 피해

## 5\. 핵심 예외 처리 및 주의사항

### 5.1. 단일 타격(합산) vs 개별 타격(분리) 판정 기준

카드 효과로 발생하는 피해의 성질에 따라 연산 및 타격 횟수가 달라집니다.

1. 조건부 추가 피해 (단일 타격 판정)  
   * 설명 : 단순히 타겟의 상태나 위치 조건을 만족했을 때 피해량이 늘어나는 경우입니다.  
   * 처리 방식 : 기본 피해량과 추가 피해량을 합산하여 한 번의 타격(1 Hit)으로 연산합니다.  
   * 예시 : 기본 4피해를 주고, 타겟이 \[표적\] 상태라면 8의 추가 피해를 줌 \-\> 기본 피해량을 12로 취급하여 Ceil(Max(0, 12 \+ 고정수치증감) \* 공격 계수 \* 방어계수) 공식에 대입  
2. 넉백 충돌 피해 (개별 타격 판정)  
   * 설명 : 카드 사용으로 대미지를 입힌 후 , 타겟이 밀려나서 (넉백) 벽이나 다른 개체와 부딪혔을 때나 공격을 여러번(다수)일 경우의 피해입니다.  
   * 처리 방식 : 스킬에 의한 기본 타격과, 충돌 시 발생하는 물리적 피해 또는 여러 여러번(다수)일 경우의 피해를 독립된 2번(2 Hit) 또는 N번(N Hit) 타격으로 분리하여 연산합니다.  
     * 예시: 넉백에 의한 피격 시퀸스:  
       1. 스킬 기본 피해 연산 및 적용 (1타)  
       2. 넉백 이동 처리  
       3. 이동 종료 지점에서 충돌 발생 시, 충돌 피해 연산 및 적용 (2타, 독립적 계산)  
     * 예시: N번 피격 시퀸스:  
       1. 스킬 기본 피해 연산 및 적용 (1타)  
       2. N번 1번 반복

### 6.2. 소수점 처리 원칙 (매우 중요)

* 게임 내의 모든 대미지 연상에서 소수점이 발생할 경우, 과정 연산(동적 피해량 산출 등)과 결과 연산 (최종 스킬/틱 피해량 산출)을 불문하고 언제나 올림(Ceil)을 사용합니다.  
* 이를 통해 최종 적용 수치가 무조건 정수로 떨어지도록 보장합니다.
