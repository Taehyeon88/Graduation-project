# 이동 시스템 기획서

최신 업데이트 : 2026.02.06  
버전 : 1.0.0 (주사위 시스템 삭제 및 선 이동 규칙 적용)  
---

## 1\. 개요

게임의 전술적 위치 선정을 담당하는 이동 시스템을 정의한다. 기존의 불확실한 주사위 이동을 배제하고, \[1칸 필수 이동\] \-\> \[카드 액션\] 이라는 확정적인 흐름을 도입한다. 플레이어는 매 턴 시작 시 반드시 위치를 변경해야 하며, 이후 카드를 통해 전략적인 추가 이동이나 공격을 수행한다.

## 2\. 그리드 및 맵 구조

### 2.1 규격

* 전체 크기 : 가로 10칸 x 세로 6칸 (총 60칸)  
* 이유:   
  * 주사위의 최대값(7\~8) 이동 시, 직선 이동뿐만 아니라 우회 기동을 할 수 있는 최소한의 공간 확보  
  * 뱀 꼬리 규칙으로 인한 퇴로 차단 시, 플레이어가 빠져나갈 구멍 제공

### 2.2 타일 상태 및 종류

각 타일은 기본적인 상태(이동 가능 / 불가) 외에 오브젝트나 효과를 가질 수 있다.  
이동 후 타일이 벽으로 변하는 ‘지난 길’ 규칙은 전술적 깊이를 위해 유지한다.

A. 기본 상태  
1\. 일반 (Normal) 

* 타일 색상 : X  
* 특징 : 아무 효과 없는 빈 땅. 이동 가능

2\. 벽 (Wall)

* 타일 색상 : X  
* 특징: 영구적인 장애물. 이동 불가

3\. 유닛 (Unit)

* 타일 색상 : X  
* 특징 : 캐릭터 또는 적이 서 있는 칸. 이동 불가

4\. 지나간 길 (Exhausted):

* 타일 색상 : 회색  
* 플레이어가 이번 턴에 밟고 지나간 타일  
* 이번 턴 종료시 까지 벽(Wall)으로 판정되어 재진입 불가  
* 턴 종료시 일반(Normal) 상태로 초기화

B. 오브젝트 및 특수 타일  
5\. 아이템 (Item):

* 타일 색상 : 파랑색  
* 특징 : 이동 가능. 플레이어가 해당 타일에 진입(Enter)하는 즉시 효과 발동 후 소멸(Despawn)  
* 예시 : 마나파편 (이번 턴에만 사용할 수 있는 마나 \+1 즉시 획득)  
* 재생성 : 다시 스폰되지 않음 (1회성)

6\. 위험 구역 (Danger Zone)

* 공격 예고 (Warning):  
  * 타일 색상 : 빨간색 & 붉은색  
  * 특징 : 몬스터의 다음 턴 공격 범위. 이동은 가능하나, 턴 종료 시 해당 위치에 머물면 데미지 받음  
* 장판 (Hazard):   
  * 타일 색상 : 해당 스킬 디자인 (플레이어 이동 모드일 시 해당 타일 테두리에 붉은색으로 표시)  
  * 특징 : 독웅덩이, 가시 함정 등. 타일에 닿는 순간 즉시 데미지나 디버프 적용

7\. 환경 지형:

* 물줄기 (Stream) : 흐르는 방향으로 강제 이동되거나, 이동력을 추가로 소모하는 등 물리적 영향을 주는 지형  
* 철도 (railway) : 일정 턴 마다 철도가 위치한 타일에 데미지를 주는 지형

## 3\. 턴 구조 및 이동 규칙

이번 업데이트의 핵심 변경 사항으로, 턴의 흐름이 \[이동 준비\]와 \[메인 행동\]으로 엄격히 분리한다.

### 3.1 턴 진행 순서

1. 턴 시작  
2. \[1단계\] 필수 이동 : 1칸 이동 수행  
3. \[2단계\] 행동 잠금 해제 : 카드 사용 권한 획득  
4. \[3단계\] 메인 행동 : 카드 사용  
5. 턴 종료

### 3.2 상세 규칙

A. \[1단계\] 필수 이동

* 조건 : 플레이어 턴이 시작되면 즉시 발동  
* 규칙:  
  * 플레이어는 상하좌우 및 대각선 중 이동 가능한 타일로 반드시 1칸 이동해야 한다.  
  * 이 단계에서는 카드를 사용할 수 없다. (패 확인은 가능하나 사용 불가 상태)  
* 제약:  
  * 만약 사방이 막혀 1칸도 움직일 수 없는 경우, **패널티(체력 감소)**를 받고 제자리에서 \[2단계\]로 넘어간다.

B. \[2단계\] 행동 잠금 해제

* 필수 이동을 완료하는 즉시, 핸드에 있는 카드의 잠금 시각 효과가 풀리며 사용 가능한 상태가 된다.

C. \[3단계\] 카드 이동

* 필수 이동 이후의 모든 이동은 **이동 효과가 있는 카드**를 사용해야만 가능하다.  
* 이동 카드 예시:  
  * 전력 질주 : 직선으로 2칸 이동.  
  * 회피 : 대각선으로 1칸 이동 후 방어도 획득

## 4\. 이동 메커니즘

### 4.1 기본 이동 방식

* 방식 : 마우스 클릭을 통한 타일 지정.  
* 판정:  
  * \[필수 이동\] 시 : 인접한 이동 가능한 8방향 타일 중 하나를 클릭하여 즉시 이동.  
  * \[카드 이동\] 시 : 카드를 드래그 후 이동 가능한 타일 중 하나를 클릭  
  * 참조 문서:  
    *  [타일 시각 효과 및 인터랙션 기획서](https://docs.google.com/document/d/1XoFpHqGQuajOiSZmWHPmtgmSlphJOF49mBtLrvKi3nU/edit?usp=sharing) *4.2. 플레이어 이동 가능 범위* 참조.  
    * [https://www.figma.com/board/qqIoS2xJb5yJQjzS7yN1Ff/%EC%A1%B8%EC%9E%91?node-id=125-175\&t=liHVEJkolOi8BRLp-1](https://www.figma.com/board/qqIoS2xJb5yJQjzS7yN1Ff/%EC%A1%B8%EC%9E%91?node-id=125-175&t=liHVEJkolOi8BRLp-1) 맵과 주사위 이미지 참조

### 4.2 타일 소모

* \[필수 이동\]으로 밟은 타일과 \[카드 이동\]으로 밟은 타일 모드 ‘**지나간 길(회색)**’이 되어, 해당 턴 동안 되돌아갈 수 없다.  
* 이는 플레이어가 “필수 이동으로 1칸 전진했다가, 이동 카드로 다시 원위치로 돌아오는” 꼼수를 방지한다.

## 5\. UI/UX 및 조작

PC 플랫폼 기준의 조작 방식

### 5.1 턴 시작 및 \[필수 이동\] 단계

* 핸드(손패) 비활성화:  
  * 턴 시작 시 뽑을 카드에서 나온 카드들이 어둡게 처리되고, 쇠사슬/자물쇠 아이콘이 덧씌워져 사용 불가함을 시각적으로 알림.  
  * 카드 위에 마우스를 올리면(마우스 호버) “이동 후 사용 가능” 툴팁 출력.  
* 이동 가이드:  
  * 캐릭터 주변 1칸(8방향) 중 이동 가능한 타일에 파랑 원 문양이 출력  
    * [타일 시각 효과 및 인터랙션 기획서](https://docs.google.com/document/d/1XoFpHqGQuajOiSZmWHPmtgmSlphJOF49mBtLrvKi3nU/edit?usp=sharing) *4.2. 플레이어 이동 가능 범위\_* 시각적 특징\_\[Synbol\] 문양 참조.  
  * 화면 상단 중앙에 “이동하여 행동 캐시”의 안내 문자 표시.

### 5.2. 행동 잠금 해제

* 1칸 이동 완료 시, 카드 잠금이 풀리는 연출과 함께 카드가 밝게 활성화됨.  
* 이후 자유롭게 카드 사용 가능.

### 5.3. 카드 이동 시

* 이동 카드를 선택하면 이동 가능한 범위가 타일 위에 표시됨.  
* 이동 불가능한 지역(벽, 지나간 길)은 표시 X
