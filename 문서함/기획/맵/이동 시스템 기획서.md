## 1\. 개요

게임 내 전술적 위치 선정을 담당하는 이동 시스템을 정의한다. 불확실한 주사위 이동을 배제하고, 플레이어가 보유한 \[카드 액션\]에 의한 확정적 이동 방식을 도입한다. 플레이어는 자신의 턴에 보유한 카드를 전략적으로 사용하여 이동하거나 공격을 수행하며, 이동한 경로는 실시간으로 제약되어 전술적 깊이를 더한다.

## 2\. 그리드 및 맵 구조

### 2.1 타일 상태 및 종류

각 타일은 기본적인 상태(이동 가능 / 불가) 외에 오브젝트나 효과를 가질 수 있다.  
이동 후 타일이 벽으로 변하는 ‘지난 길’ 규칙은 전술적 깊이를 위해 유지한다.

A. 기본 상태  
1\. 일반 (Normal) 

* 타일 색상 : X  
* 특징 : 아무 효과 없는 빈 땅. 이동 가능

2\. 벽 (Wall)

* 타일 색상 : X  
* 특징: 영구적인 장애물. 이동 불가

3\. 유닛 (Unit)

* 타일 색상 : X  
* 특징 : 캐릭터 또는 적이 서 있는 칸. 이동 불가

B. 오브젝트 및 특수 타일  
5\. 아이템 (Item):

* 타일 색상 : 파랑색  
* 특징 : 이동 가능. 플레이어가 해당 타일에 진입(Enter)하는 즉시 효과 발동 후 소멸(Despawn)  
* 예시 : 마나파편 (이번 턴에만 사용할 수 있는 마나 \+1 즉시 획득)  
* 재생성 : 다시 스폰되지 않음 (1회성)

6\. 위험 구역 (Danger Zone)

* 공격 예고 (Warning):  
  * 타일 색상 : 빨간색 & 붉은색  
  * 특징 : 몬스터의 다음 턴 공격 범위. 이동은 가능하나, 턴 종료 시 해당 위치에 머물면 데미지 받음  
* 장판 (Hazard):   
  * 타일 색상 : 해당 스킬 디자인 (플레이어 이동 모드일 시 해당 타일 테두리에 붉은색으로 표시)  
  * 특징 : 독웅덩이, 가시 함정 등. 타일에 닿는 순간 즉시 데미지나 디버프 적용

7\. 환경 지형:

* 물줄기 (Stream) : 흐르는 방향으로 강제 이동되거나, 이동력을 추가로 소모하는 등 물리적 영향을 주는 지형  
* 철도 (railway) : 일정 턴 마다 철도가 위치한 타일에 데미지를 주는 지형

## 3\. 이동 메커니즘

### 3.1 기본 이동 방식

* 방식 : 마우스 클릭을 통한 타일 지정.  
* 판정:  
  * 플레이어가 이동 성질을 가진 카드를 드래그하여 사용한 후, 하이라이트된 이동 가능 범위 내의 타일을 클릭하여 이동한다.

PC 플랫폼 기준의 조작 방식

### 4.1. 카드 이동 시

* 이동 카드를 선택하면 이동 가능한 범위가 타일 위에 표시됨.  
* 이동 불가능한 지역(벽, 지나간 길)은 \[이동 표시\] 문양이 나타나지 않음
